## NP Simulation Requirements

Every NP simulation must include:

- Assessment
- Diagnosis
- Differential diagnosis
- Management
- Prescribing relevance where in scope
- Follow-up
- Patient education
- Escalation or referral threshold

## Build Sequence

1. Add certification mapping to existing NP simulations.
2. Map each NP simulation to diagnostic reasoning, management, prescribing safety, follow-up, and patient education.
3. Complete CNPLE first because it is the strongest current NP revenue pathway.
4. Complete FNP second because it has the strongest US NP commercial potential.
5. Build AGPCNP, PMHNP, WHNP, and PNP-PC after shared NP simulation standards are enforced.
