# Global Publication Readiness Report

## Publication Gate

An item may be marked `published` only after completing:

1. Draft
2. Internal Review
3. Clinical Review
4. Educational Review
5. SEO Review
6. Ready For Publication
7. Published

Localized content must also pass localization review.

## Required Pass Conditions

| Condition | Required |
| --- | --- |
| Overall quality score | `90+` |
| Clinical review | Passed |
| Educational review | Passed |
| SEO review | Passed for public/indexable content |
| Accessibility review | Passed |
| Localization review | Passed for localized content |
| Duplicate review | Cleared if similarity `90%+` |
| Blueprint metadata | Present when applicable |
| Adaptive routing | Present for premium learning assets |
| Rationale | Complete when applicable |
| Hint | Complete when applicable |
| Clinical pearl | Complete when applicable |
| References | Present for authority content |

## Blockers

The following always block publication:

- Placeholder content
- Thin scaffold pages
- AI filler
- Clinical inaccuracies
- Weak rationales
- Implausible distractors
- Direct-answer hints
- Generic clinical pearls
- Definition-only flashcards
- Missing blueprint mapping
- Missing review status
- Unresolved duplicate flags

## Decision Labels

| Label | Meaning |
| --- | --- |
| Draft | Not reviewed |
| Internal Review | Content team reviewing structure |
| Clinical Review | Clinician reviewing accuracy and safety |
| Educational Review | Educator reviewing teaching value |
| SEO Review | Public page reviewed for search and metadata |
| Ready For Publication | All gates passed, not yet published |
| Published | Live and governed |
| Archived | Removed from active learner/public use |
