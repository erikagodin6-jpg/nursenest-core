## Application Timeline Template

- research: 12-18 months before start — Compare programs, locations, prerequisites, tuition, clinical placement expectations, and licensing pathway.
- prerequisites: 9-15 months before start — Complete required courses and plan grade improvement if competitive averages are above current marks.
- application: 6-12 months before start — Prepare application portal requirements, transcripts, program choices, fees, and deadlines.
- casper: 2-6 months before deadline — Practice ethical reasoning, communication, reflection, and timed written responses if CASPER is required.
- interview: 1-4 months before decision — Prepare behavioural examples, healthcare motivation, teamwork examples, and program-specific questions.
- references: 1-3 months before deadline — Request references early and provide each referee with program context and submission instructions.
- decision: Offer period — Compare offers, clinical placement requirements, commuting, cost, support services, and licensing goals.
