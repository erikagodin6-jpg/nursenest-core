# Blog batch 1 — static long-tail import and publish

**Date:** 2026-05-09 (content `publishedAt` / `updatedAt`)  
**Scope:** Hybrid static long-tail pipeline (`src/content/blog-static-longtail/` + validators + loader). DB CMS unchanged.

## Summary

- **10 target slugs:** 2 already existed on the same topics (updated with extra internal links only); **8 new** markdown files added.
- **Draft workflow:** `draft: true` in frontmatter is now parsed and **excludes** posts from `listBlogStaticLongtailRecords()`, public supplement merge, sitemap static rows, and `getBlogStaticLongtailRecord()` until `draft` is omitted or set `false`. **`npm run validate:blog-static-longtail` validates all files** (including drafts) via `listAllBlogStaticLongtailFileRecords()`.
- **Truthpack:** `.vibecheck/truthpack/` not present in this workspace; no truthpack fields asserted.
- **Admin preview:** Static long-tail is served as synthetic `BlogPost` rows from markdown when the live DB has no row for the slug (`publishedBlogPostFromLongtailRecord`); there is **no separate admin CMS preview** for these files. Verify locally at `/blog/<slug>` when the app is running and the slug is not overridden by a live DB post.
