# Duplicate Content Opportunity Report

Date: 2026-05-31

Status: internal duplication prevention report. This report supersedes ad hoc duplication assumptions but does not publish content.

## Summary

The largest duplication risk is rebuilding global clinical science for every country. Existing content strongly supports reuse for lessons, flashcards, clinical skills, ECG, labs, pharmacology, and simulations. The largest rewrite/new-content need is exam-specific question format, regulator language, scope, and health-system context.

## Deduplication Rules

1. Never create country-specific copies of global clinical science unless the clinical standard differs.
2. Store global topics once and attach country/exam overlays.
3. Questions may duplicate a clinical concept only when the exam format, regulator context, or scope changes.
4. Flashcards should inherit from global decks with country-specific inclusion/exclusion tags.
5. Simulations should reuse the same clinical state engine with country-specific escalation language.
