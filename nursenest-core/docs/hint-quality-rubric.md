# Hint Quality Rubric

Date: 2026-05-31
Status: Question, hint, rationale, and clinical pearl quality overhaul

Future content flags: `published=false`, `launchReady=false`, `adminOnly=true` until reviewed.

## Purpose

Hints should guide learner reasoning without revealing the answer. A strong hint helps learners notice the correct clinical frame, not the correct letter.

## Rubric

| Score | Standard | Description |
| ---: | --- | --- |
| 5 | Excellent | Guides reasoning through clinical cues, urgency, mechanism, scope, or patient safety without disclosing the answer |
| 4 | Good | Points to the correct decision framework but could be more specific |
| 3 | Adequate | Gives a general study direction but may not sharpen clinical reasoning |
| 2 | Weak | Vague, generic, or mostly restates the stem |
| 1 | Unsafe | Reveals the answer, names the option, or misleads clinical reasoning |

## Good Hint Patterns

| Pattern | Example |
| --- | --- |
| Priority frame | Consider which finding represents the greatest immediate threat to airway, breathing, or circulation. |
| Mechanism frame | Think about which intervention addresses the underlying pathophysiology. |
| Scope frame | Ask which action requires nursing judgment rather than task completion. |
| Trend frame | Compare the current finding with the previous trend before choosing an action. |
| Safety frame | Identify which option prevents the most serious near-term patient harm. |
| Exam frame | Decide whether the item is asking for first action, best teaching, delegation, or evaluation. |

## Rejected Hint Patterns

| Pattern | Example | Reason |
| --- | --- | --- |
| Option disclosure | Review option B. | Reveals answer location |
| Answer wording | The correct answer focuses on oxygen. | Gives away the answer concept |
| Generic command | Think carefully. | Does not guide reasoning |
| Rationale preview | The answer is about sepsis. | Reveals diagnosis |
| Unhelpful recall | Review the chapter. | No clinical reasoning support |

## Required Hint Elements

Each hint should include at least one:

1. Clinical cue direction
2. Priority framework
3. Pathophysiology or mechanism prompt
4. Scope or role prompt
5. Safety or escalation prompt
6. Exam decision-type prompt

## Pathway-Specific Hint Examples

| Pathway | Good Hint |
| --- | --- |
| RN | Consider whether this is an acute change that should be addressed before routine teaching or documentation. |
| RPN / PN | Think about whether the finding is expected and predictable or requires reporting and reassessment. |
| NP | Consider which diagnosis or management step is safest based on the red flags in the history. |
| Pre-Nursing | Think about the primary function of this structure before choosing the answer. |
| Admissions | Eliminate choices that do not answer the exact question being asked before calculating. |
| Allied Health | Consider the profession-specific safety risk before selecting the next action. |
| ECG | Use rate, regularity, P waves, PR interval, and QRS width before naming the rhythm. |
| Labs | Decide whether the abnormal value is isolated or part of a dangerous pattern. |
| Medication Math | Set up the units first, then check whether the final answer is clinically reasonable. |
| Pharmacology | Consider contraindications and monitoring before choosing an administration or teaching action. |

## Publish Gate

Hints fail quality review when they:

- Reveal the answer.
- Name an answer option.
- Use only generic study advice.
- Do not connect to the stem.
- Encourage unsafe or outside-scope reasoning.
