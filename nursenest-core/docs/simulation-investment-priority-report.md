## Executive Allocation

1. Fund simulation-to-pathway and simulation-to-topic mapping first because readiness cannot be scored without it.
2. Build RN and RPN/PN simulation parity before future academy simulation expansion.
3. Complete high-risk loop simulations before lower-risk topic simulations.
4. Expand ECG deterioration, pharmacology safety, and lab interpretation because they create reusable simulation logic across RN, PN/RPN, NP, and future institutional products.
5. Keep future academy simulation work hidden until core revenue pathways pass simulation readiness gates.
