## NP Build Sequence

1. Close CNPLE question and flashcard gaps first because CNPLE already exceeds the lesson minimum.
2. Close FNP lesson and question gaps next because FNP has the strongest current flashcard-derived availability among US NP pathways.
3. Build AGPCNP and PMHNP in parallel only after diagnostic reasoning, prescribing relevance, and follow-up planning standards are enforced.
4. Build WHNP and PNP-PC after core NP enrichment patterns are stable.
5. Do not create ENP content until an evidence-backed ENP pathway registry exists.

## NP Quality Requirements

Every NP expansion item must include diagnostic reasoning, differential diagnosis, guideline-informed management, prescribing relevance where in scope, follow-up planning, clinical judgment, blueprint mapping, and flashcard output.
