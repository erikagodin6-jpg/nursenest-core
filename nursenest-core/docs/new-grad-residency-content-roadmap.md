## Product Positioning

The New Graduate Nurse Residency product should feel like a bridge between nursing school and safe first-year practice. It should help newly licensed nurses translate exam knowledge into shift organization, prioritization, escalation, communication, documentation, medication safety, and confidence in real clinical environments.

Primary promise: Pass the exam. Thrive in practice.

## Lesson Blueprint

Every lesson should include:

| Section | Purpose |
| --- | --- |
| Learning objectives | Define the practice behavior the learner should demonstrate |
| Transition-to-practice context | Explain where this appears during a real shift |
| Core concepts | Teach the clinical, professional, or workflow principle |
| Decision-making model | Show how a safe nurse thinks through the situation |
| Patient safety risks | Identify what can go wrong if the issue is missed |
| Communication language | Provide scripts for escalation, handoff, and team communication |
| Documentation standard | Show what must be charted and why |
| Clinical pearls | Give memorable practice insight |
| Reflection prompt | Build professional judgment and confidence |
| Knowledge check | Verify application, not recall only |
| Related content | Link to questions, simulations, clinical skills, medication safety, and readiness domains |

## Question Framework

Questions should test:

| Skill | Example Focus |
| --- | --- |
| Prioritization | Which patient should be assessed first at shift start |
| Escalation | When to notify the provider, charge nurse, rapid response, or code team |
| Delegation | What may be delegated and what requires RN judgment |
| Documentation | What must be documented after a safety event |
| Communication | How to respond to conflict, family concerns, or unclear orders |
| Clinical judgment | How to interpret subtle deterioration and choose the next action |

Each question must include hint, correct answer, why correct, why incorrect, clinical application, clinical pearl, exam or practice strategy, and related content.
