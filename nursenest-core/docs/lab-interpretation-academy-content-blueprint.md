## Product Positioning

Lab Interpretation Academy should teach learners how to interpret abnormal results clinically, not memorize reference ranges in isolation. The product should connect lab patterns to patient assessment, urgency, escalation, safety, pharmacology, and likely complications.

## Interpretation Model

Every lab activity should teach:

1. What is abnormal
2. Why it may be abnormal
3. How it fits the assessment
4. What trend matters
5. What is the immediate safety risk
6. What should be done next
7. When to escalate
8. What complication may occur if missed

## Case Types

| Case Type | Examples |
| --- | --- |
| Pattern recognition | DKA, sepsis, renal failure, GI bleed, heart failure |

## Readiness Domains

| Domain | Signals |
| --- | --- |
| Critical Value Recognition | Identifies urgent abnormal results |
| Pattern Interpretation | Connects multiple lab findings to likely clinical problem |
| Trend Analysis | Recognizes worsening or improving trajectory |
| Escalation | Communicates urgent results and appropriate concern |
| Medication Monitoring | Links lab results to medication safety decisions |
| Clinical Integration | Combines labs with assessment and patient context |

## Global Reuse Strategy

Target reusable global content: 80-90%.

Clinical interpretation is broadly reusable. Country-specific supplements should handle reporting standards, units, critical value policy, and scope expectations.
