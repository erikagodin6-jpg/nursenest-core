## Purpose

Rationales are a primary teaching surface. They should feel like an experienced clinician explaining how to think, not a system defending an answer key.

## Required Structure

Every question rationale must include:

1. Correct Answer
2. Why It Is Correct
3. Why The Other Options Are Incorrect
4. Clinical Reasoning
5. Patient Safety Implications
6. Exam Strategy
7. Clinical Application
8. Clinical Pearl
9. Related Topics

## Section Standards

| Section | Standard |
| --- | --- |
| Correct Answer | Names the correct response clearly |
| Why It Is Correct | Connects the answer to the stem-specific cue |
| Why The Other Options Are Incorrect | Explains why each distractor is tempting but wrong, late, unsafe, outside scope, or unsupported |
| Clinical Reasoning | Teaches the decision process |
| Patient Safety Implications | Names the harm that could occur if missed |
| Exam Strategy | Explains how to approach similar items |
| Clinical Application | Connects to real bedside or professional practice |
| Clinical Pearl | Gives a memorable insight beyond restating the rationale |
| Related Topics | Links the learner to remediation and deeper study |

## Clinical Pearl Standard

Clinical pearls should be high yield, memorable, and practice relevant.

Good: Patients with worsening heart failure often gain weight before developing obvious respiratory symptoms.

Bad: Heart failure can cause fluid overload.

## Distractor Analysis Standard

Every wrong answer should map to a reason:

| Distractor Problem | Example Explanation |
| --- | --- |
| Wrong timing | This may be appropriate after stabilization, but it delays the first priority. |
| Wrong role | This requires provider-level decision-making or exceeds the learner's role. |
| Wrong patient | This action fits a stable patient, not a patient with acute deterioration. |
| Wrong priority | This addresses comfort or teaching before airway, breathing, circulation, or safety. |
| Unsupported by stem | The stem does not provide evidence for this intervention. |
