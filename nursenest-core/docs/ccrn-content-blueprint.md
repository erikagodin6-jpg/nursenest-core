## Product Positioning

CCRN Academy should prepare critical care nurses for certification while strengthening ICU clinical judgment. It should combine exam preparation with practical readiness for hemodynamic instability, ventilator management, shock states, critical pharmacology, ECG interpretation, ABG interpretation, and high-risk deterioration.

## Lesson Requirements

Each lesson should include pathophysiology, ICU assessment, diagnostics, monitoring priorities, first actions, escalation triggers, pharmacology considerations, safety risks, clinical pearls, exam relevance, and a knowledge check.

High-risk topics require clinical review before publication consideration:

| High-Risk Area | Review Focus |
| --- | --- |
| Vasopressors | Dosing concepts, monitoring, extravasation response, titration safety |
| Ventilation | Modes, alarms, oxygenation vs ventilation, extubation risk |
| Shock | Differentiation, early interventions, fluid and pressor decisions |
| ECG emergencies | VT, VF, STEMI, heart blocks, unstable tachyarrhythmias |
| ABGs | Compensation, ventilator response, urgent escalation |

## Question Framework

Question distribution should emphasize analysis and clinical judgment:

| Type | Target Share |
| --- | ---: |
| Clinical judgment MCQ | 35% |
| Prioritization and escalation | 20% |
| Hemodynamic interpretation | 15% |
| Case-based questions | 15% |
| ECG and ABG reasoning | 10% |
| Recall support | 5% |

Every rationale must explain the critical care consequence of delayed or incorrect intervention.

## Global Reuse Strategy

Target reusable global content: 75-85%.

Country-specific supplements should cover scope, medication naming, delegation expectations, local protocols, and documentation norms. Exam-specific supplements should align terminology and weighting to certification blueprints.
