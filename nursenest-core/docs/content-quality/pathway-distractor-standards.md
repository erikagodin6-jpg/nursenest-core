# Pathway Distractor Standards

Status: Governance standard

## RN

RN distractors must test assessment, prioritization, escalation, delegation, medication safety, teaching, and documentation. Wrong answers should often be partially correct but unsafe by timing, priority, or scope.

Required mappings:

- Assessment error
- Priority error
- Under-escalation or failure-to-rescue
- Communication error for provider/team notification

## RPN / PN

RPN/PN distractors must preserve scope. They should test predictable versus unstable care, reporting, medication administration safety, reassessment, documentation, and delegation boundaries.

Required mappings:

- Scope error
- Assessment error
- Timing error
- Medication-monitoring failure

## NP

NP distractors must be advanced-practice depth. They should test diagnosis, differential diagnosis, guideline-based management, prescribing relevance, referral thresholds, follow-up, and red flags.

Required mappings:

- Premature closure
- Anchoring bias
- Pattern-recognition error
- Medication-monitoring failure

## ECG

ECG distractors must test recognition, interpretation, hemodynamic impact, medication implications, escalation, and deterioration pathways.

Required mappings:

- Interpretation error
- Pattern-recognition error
- Trend-interpretation failure
- Failure-to-rescue when deterioration is missed

## Labs

Lab distractors must test pattern recognition, trend analysis, critical values, medication monitoring, and escalation.

Required mappings:

- Interpretation error
- Trend-interpretation failure
- Pattern-recognition error
- Under-escalation for critical values

## Pharmacology

Pharmacology distractors must test monitoring, contraindications, adverse effects, hold parameters, interactions, patient teaching, and therapeutic response.

Required mappings:

- Medication-monitoring failure
- Safety error
- Scope error for prescribing or independent dose changes
- Communication error for reportable findings
