# Distractor Publish Gate Expansion

Status: Enforcement standard

## Blocking Rules

A question is blocked from publication when any distractor has:

- Score below `80`
- Missing misconception mapping
- Missing why-tempting analysis
- Missing why-incorrect analysis
- Missing safety or consequence analysis
- Missing remediation mapping
- Missing readiness-domain mapping
- Duplicate or throwaway distractor text
- Unrealistic clinical workflow

## Score Bands

| Score | Gate | Rule |
| ---: | --- | --- |
| 0-79 | Fail or review | Not publishable |
| 80-94 | Publish eligible | Requires normal review |
| 95-100 | Flagship | Eligible for flagship distractor library |

## Required Wrong-Answer Explanation

Every incorrect option must answer:

1. Why a learner selected it.
2. Why it seems reasonable.
3. Why it is wrong.
4. What risk it creates.
5. What misconception it represents.
6. How to remediate it.

## Enforcement Surfaces

The canonical scorer is `src/lib/questions/distractor-quality-score.ts`.

The publication enforcement contract blocks low distractor quality through `src/lib/questions/question-publication-enforcement-contracts.ts`.

The source-backed audit is generated by `scripts/audit-distractor-quality.mts`.
