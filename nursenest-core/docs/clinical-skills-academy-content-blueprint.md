## Product Positioning

Clinical Skills Academy should become a competency-based practice environment for assessment, procedures, communication, documentation, medication safety, patient education, and emergency skills. It should integrate with the Competency Passport so learners demonstrate readiness through repeated performance, not participation alone.

## Skill Template

Every skill should include:

| Section | Purpose |
| --- | --- |
| Indications | When the skill is needed |
| Contraindications or cautions | When the skill requires extra safety checks |
| Equipment | What is needed before starting |
| Preparation | Patient, environment, safety, and infection prevention |
| Procedure steps | Ordered, competency-aligned actions |
| Clinical reasoning | Why each key step matters |
| Communication | What to explain to the patient or team |
| Documentation | What must be recorded |
| Complications | What can go wrong and what to do |
| Competency evidence | Knowledge, simulation, documentation, and repeated success signals |

## Competency Passport Integration

Each skill should map to:

| Evidence Type | Requirement |
| --- | --- |
| Knowledge | Lesson and question performance |
| Reasoning | Scenario and decision-point performance |
| Simulation | Active workflow completion |
| Documentation | Accurate charting or note completion |
| Communication | Patient education, SBAR, or team interaction |
| Repetition | Multiple successful attempts for high-risk skills |

## Simulation Tracks

| Track | Examples |
| --- | --- |
| Assessment | Respiratory assessment, neuro checks, pain assessment |
| Medication safety | Rights of medication administration, high-alert meds, adverse reaction |
| IV therapy | Infiltration, phlebitis, compatibility, pump programming |
| Wound care | Pressure injury staging, sterile dressing change, infection signs |
| Documentation | Fall note, medication refusal, patient education note |
| Communication | Provider notification, patient refusal, difficult family interaction |
| Emergency skills | Rapid response, code blue role, airway support, bleeding control |

## Global Reuse Strategy

Target reusable global content: 75-85%.

Core technique and safety principles are reusable. Country-specific supplements should address scope, documentation standards, terminology, and institution policy variations.
