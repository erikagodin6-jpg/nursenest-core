## Product Positioning

Preceptor & Clinical Educator Academy should prepare nurses, allied health professionals, faculty, and clinical educators to teach, coach, evaluate, document, and support learners in clinical environments. It should support future enterprise, school, and hospital offerings while remaining hidden until launch strategy is approved.

## Faculty Scenario Standards

Scenarios should include:

| Element | Purpose |
| --- | --- |
| Learner context | Experience level, role, and learning need |
| Clinical context | Patient-care or classroom situation |
| Educator decision point | What the preceptor or educator should do next |
| Feedback language | Example words that are direct, respectful, and actionable |
| Documentation requirement | What must be recorded and why |
| Escalation threshold | When unsafe performance requires intervention |
| Reflection | How to support growth without lowering standards |

## Readiness Domains

| Domain | Signals |
| --- | --- |
| Teaching Readiness | Uses clear, learner-centered instruction |
| Feedback Readiness | Gives specific, timely, behavior-based feedback |
| Coaching Judgment | Balances support with safe autonomy |
| Competency Evaluation | Uses evidence, rubrics, and repeated demonstration |
| Remediation Planning | Identifies gaps and assigns targeted learning |
| Professional Documentation | Creates fair, defensible educational records |

## Global Reuse Strategy

Target reusable global content: 75-85%.

Teaching and coaching principles are reusable. Country-specific and institution-specific supplements should cover program policies, legal documentation standards, regulatory language, and placement requirements.
