# NP Core Library Roadmap

Date: 2026-05-31

Status: internal content roadmap. Core inventory may be reused by active pathways, but unreleased future certification content remains hidden.

## Objective

Create a reusable NP Core Library that supports every NP certification without duplicating advanced-practice fundamentals.

## Core Content Targets

| Content Type | Target |
| --- | ---: |
| Lessons | 500+ |
| Flashcards | 5,000+ |
| Questions | 3,000+ |
| Clinical Cases | 200+ |

## Core Domains

### Advanced Assessment

Focus:

- Comprehensive history and physical exam.
- Red flag recognition.
- Focused systems assessment.
- Diagnostic uncertainty.
- Age-appropriate assessment.

Reused by all NP certifications.

### Diagnostics

Focus:

- Lab selection and interpretation.
- Imaging selection.
- Differential diagnosis.
- Diagnostic stewardship.
- Follow-up thresholds.

Reused by all NP certifications with specialty overlays.

### Clinical Decision-Making

Focus:

- Prioritization.
- Risk stratification.
- Shared decision-making.
- Escalation and referral.
- Safety-netting.

Reused across primary care, mental health, pediatrics, acute care, emergency, and CNPLE pathways.

### Evidence-Based Practice

Focus:

- Guideline appraisal.
- Research interpretation.
- Quality improvement.
- Risk/benefit reasoning.
- Patient-centered application.

### Advanced Pharmacology

Focus:

- Mechanisms.
- Prescribing safety.
- Contraindications.
- Monitoring.
- Adverse effects.
- Drug interactions.
- Deprescribing.

Specialty overlays handle psychopharmacology, pediatrics, pregnancy/lactation, ICU, emergency, and Canadian prescribing context.

### Advanced Pathophysiology

Focus:

- Disease mechanisms.
- Multisystem reasoning.
- Clinical progression.
- Complications.
- Diagnostic patterns.

### Health Promotion

Focus:

- Screening.
- Prevention.
- Immunization.
- Counseling.
- Population health.
- Equity.

### Professional Issues, Ethics, Leadership, And Research

Focus:

- Scope.
- Ethics.
- Documentation.
- Interprofessional collaboration.
- Leadership.
- Research and QI.
- Legal and regulatory context.

## Reuse Rule

The core library teaches advanced-practice fundamentals. Certification layers teach role-specific application.

Do not create a separate FNP, PMHNP, WHNP, PNP, or CNPLE copy of a core topic unless the specialty standard materially changes the clinical reasoning.
