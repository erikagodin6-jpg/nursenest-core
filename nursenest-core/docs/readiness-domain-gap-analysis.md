## Required Readiness Domains

Every pathway must report:

- Clinical readiness
- Safety readiness
- Reasoning readiness
- Simulation readiness
- Professional readiness

For launch-candidate maturity, each domain must reach at least 90%; for RN and RPN/PN commercial maturity, target is 95%+.

## Domain Definitions

| Domain | Evidence Required |
| --- | --- |
| Clinical readiness | Lessons, questions, cases, simulations, clinical skills mapped to clinical priorities |
| Safety readiness | Medication safety, escalation, red flags, adverse event prevention, high-risk topics |
| Reasoning readiness | NCJMM or equivalent reasoning stages, rationales, prioritization, trend interpretation |
| Simulation readiness | Scenario coverage, debriefs, outcome scoring, remediation triggers, competency updates |
| Professional readiness | Documentation, communication, delegation, ethics, scope, accountability |

## Missing Domain Infrastructure

| Missing Infrastructure | Impact |
| --- | --- |
| Pathway-level readiness domain registry | Prevents readiness scoring by exam and profession |
| Topic-level readiness mapping | Prevents high-risk topic maturity claims |
| Simulation-to-readiness score updates | Prevents simulation coverage from improving learner profile maturity |
| Flashcard-to-readiness links | Prevents weak-area remediation from routing through retention tools |
| Professional practice loop mapping | Weakens documentation, communication, delegation, and scope maturity |

## Roadmap

| Stage | Required Work | Completion Evidence |
| --- | --- | --- |
| Domain registry | Add canonical readiness domains to pathway dashboards | Every pathway has five domain scores |
| Asset mapping | Map lessons, questions, flashcards, cases, simulations, skills to readiness domains | Asset-level domain coverage reports |
| Simulation scoring | Feed simulation analytics into readiness domains | Simulation outcomes update competency profile |
| High-risk loops | Assign high-risk topics to clinical/safety/reasoning/simulation domains | No high-risk topic is domain-incomplete |
| Professional readiness | Complete documentation, communication, delegation, ethics, scope loops | Professional readiness reaches 90%+ |
