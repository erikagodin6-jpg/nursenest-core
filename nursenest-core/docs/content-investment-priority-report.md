1. Allocate the next content sprint to RN and RPN/PN question completion because those are the top two current revenue-pathway gaps.
2. Run flashcard status instrumentation before claiming flashcard publication readiness.
3. Build simulations and high-risk loops around the same priority topics so question, flashcard, simulation, and remediation assets converge.
4. Complete CNPLE and FNP before expanding lower-readiness NP specialties.
5. Keep future academies paused until RN readiness exceeds 95%, RPN readiness exceeds 95%, and NP readiness exceeds 90% by exact evidence.
