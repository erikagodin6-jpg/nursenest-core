# Monetization Readiness

Status: Governance only. No pricing route or entitlement change is authorized by this document.

## Required Evidence

Every future academy must document:

- Subscription mapping
- Tier mapping
- Upgrade paths
- Proration model
- Institutional licensing
- Free trial structure
- Retention strategy
- Referral strategy
- Coupon strategy
- Reporting strategy
- Refund/support policy impact
- Revenue attribution model
- Conversion funnel events
- Renewal protection plan

## Monetization Gate

Commercial launch requires 100% monetization readiness.

| Area | Evidence | Blocking Condition |
|---|---|---|
| Subscription mapping | Product-to-plan table. | Product access cannot be resolved from plan. |
| Tier mapping | Free, trial, paid, institutional features. | User cannot understand what is included. |
| Upgrade paths | CTA, checkout, confirmation, return route. | Paid path untested. |
| Proration model | Upgrade/downgrade behavior. | Billing ambiguity. |
| Institutional licensing | Seat, cohort, reporting, admin rules. | School use case unsupported. |
| Free trial | Trial length, limits, success metric. | Trial exposes premium without conversion plan. |
| Retention | Reminders, progress, value reinforcement. | No renewal strategy. |
| Referral | Creator/student/institution strategy. | Not required for beta; required for scale. |
| Coupons | Eligibility and expiration rules. | Uncontrolled discounting. |
| Reporting | Revenue by product, plan, cohort, source. | No attribution. |

## Monetization Readiness Score

- 0-59: Not commercially usable
- 60-79: Pricing concept only
- 80-89: Beta monetization candidate
- 90-99: Nearly ready, unresolved blocker exists
- 100: Launch eligible
