## Phase 10: Competency Passport Expansion

Every skill should produce a multi-evidence readiness profile. Participation alone is not competency.

| Evidence Type | Required Evidence | Progression Model |
|---|---|---|
| Knowledge evidence | Lesson completion, question performance, rationale review. | Not started -> learned -> validated. |
| Reasoning evidence | Case decisions, prioritization, complication recognition. | Cue recognition -> interpretation -> safe decision. |
| Simulation evidence | Completed workflow, decision points, outcome evaluation. | Observed -> assisted -> independent. |
| Documentation evidence | Notes, flowsheets, incident/education/escalation documentation. | Incomplete -> acceptable -> defensible. |
| Communication evidence | SBAR, patient teaching, family/team communication. | Scripted -> clear -> confident. |
| Repetition evidence | Multiple successful attempts for high-risk skills. | Single pass -> repeated success -> maintenance due. |

### Progression Statuses

- Not Started
- Learning
- Practice Required
- Supervised Ready
- Competency Candidate
- Competency Demonstrated
- Maintenance Due

## Phase 12: Global Reuse Strategy

Clinical skills are broadly reusable, but scope, documentation, terminology, equipment policy, and institution workflows require overlays.

| Asset Category | Classification | Expected Reuse Rate | Supplement Needed |
|---|---|---:|---|
| Core skill rationale | Global | 85-90% | Language/local terminology only. |
| Procedure steps | Global with scope overlay | 75-85% | Role scope, local policy, equipment variations. |
| Infection prevention | Global with country overlay | 80-90% | Local standards and isolation terminology. |
| Documentation exercises | Country/institution-specific overlay | 55-70% | Legal/documentation standards and EHR language. |
| Communication scripts | Global with culture/context overlay | 70-80% | Local role language and escalation pathways. |
| Emergency response workflows | Global with certification/scope overlay | 65-80% | Local emergency protocols and role expectations. |
| OSCE-style encounters | Exam/institution-specific overlay | 55-75% | Rubric, station format, scope, documentation expectations. |
| Competency reports | Institution-specific | 45-65% | Thresholds, report structure, privacy, roster workflows. |

Reuse goal:

- Global clinical skills core: 75-85%.
- Country/institution overlays: only where scope, documentation, terminology, equipment, or policy differs.
- Future allied health overlays: reuse common safety, communication, documentation, mobility, infection prevention, and emergency response foundations with profession-specific scope.
