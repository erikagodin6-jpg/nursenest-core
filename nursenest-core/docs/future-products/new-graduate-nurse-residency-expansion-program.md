## Phase 8: Clinical Judgment Residency Engine

Every lesson, question, case, and simulation should map to the residency judgment cycle:

```txt
Recognize -> Interpret -> Prioritize -> Act -> Evaluate -> Reflect
```

| Judgment Domain | Required Evidence |
|---|---|
| Recognize | Cues, red flags, baseline comparison, trend detection. |
| Interpret | Why findings matter, pathophysiology, risk meaning, differential possibilities where appropriate. |
| Prioritize | ABCs, safety, unstable-before-stable, scope, time sensitivity. |
| Act | Scope-appropriate interventions, communication, escalation, documentation. |
| Evaluate | Patient response, reassessment, outcome measures, follow-up. |
| Reflect | What went well, what was missed, how to prevent repeat errors. |

### Mapping Requirement

No residency asset is complete unless it includes:

- Primary judgment domain
- Secondary judgment domain
- Patient safety risk
- Communication requirement
- Documentation requirement
- Escalation trigger where applicable
- Reflection prompt
