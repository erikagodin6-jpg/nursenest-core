# Flagship Question Standard

Date: 2026-06-01
Status: Top 10% premium question library standard

## Definition

Flagship questions are the top 10% of NurseNest questions. They should demonstrate the platform's highest clinical reasoning, distractor quality, rationale depth, and learner-teaching value.

## Required Threshold

| Requirement | Minimum |
| --- | ---: |
| Overall V2 question score | 95+ |
| Rationale quality | 95+ |
| Hint quality | 95+ |
| Clinical pearl quality | 95+ |
| Distractor quality | 95+ |
| Related-content quality | 90+ |

## Required Qualities

| Quality | Standard |
| --- | --- |
| Complex clinical reasoning | Requires cue interpretation, prioritization, trend recognition, differential thinking, or scope judgment |
| Strong distractors | Each wrong option reflects a plausible clinical misconception |
| High-fidelity rationale | Explains cue interpretation, why correct, why incorrect, safety implications, and bedside application |
| Clinical pearl | Memorable, practice-ready, and useful beyond the item |
| Related-content integration | Links to lessons, flashcards, simulations, labs, ECG, pharmacology, or skills |
| Readiness scoring integration | Updates clinical, safety, reasoning, simulation, or professional readiness domains |

## Flagship Library Uses

Flagship questions may be used for:

- Premium learner examples
- Review-mode demonstrations
- Marketing screenshots after approval
- Faculty/institutional demos
- Daily question campaigns
- Adaptive study calibration anchors
- CAT item-quality weighting

## Exclusions

Do not include a question in the flagship library if it:

- Requires unresolved enhanced clinical review
- Has a generic rationale
- Has an answer-revealing hint
- Has weak or obvious distractors
- Lacks related remediation content
- Has no clinical application
- Is duplicated or near-duplicated
