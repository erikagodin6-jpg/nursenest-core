## Required Case Types

| Case Type | Purpose |
| --- | --- |
| NGN cases | Teach NCJMM cue recognition, analysis, prioritization, action, evaluation |
| Branching cases | Show consequences of nursing choices and escalation timing |
| Longitudinal cases | Teach trend recognition, handoff, shift-to-shift thinking, disease progression |
| Interprofessional cases | Teach role clarity, team communication, consultation, discharge planning |
| Deterioration cases | Teach early warning signs, rescue actions, and safety response |

## Case Library Completion Rules

Every mature high-risk topic requires:

- At least one NGN case
- At least one simulation-linked deterioration case
- A debrief
- Clinical reasoning explanation
- Priority nursing actions
- Flashcard extraction
- Remediation mapping
- Readiness domain mapping
