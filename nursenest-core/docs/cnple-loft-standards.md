# CNPLE LOFT Standards

## Terminology
- Use LOFT Simulation for CNPLE learner flows.
- Do not use CAT terminology for CNPLE.

## Learner Hub
CNPLE learner flows should expose:
- Practice Tests
- LOFT Simulation
- Flashcards
- Lessons
- Question Bank

## Simulation Requirements
CNPLE simulations should support:
- evolving patient state
- diagnostics
- prescribing
- follow-up visits
- SOAP note interpretation
- differential diagnosis

## Content Requirements
CNPLE questions require:
- rationale
- Canadian NP framing
- competency tagging
- body-system tagging
- duplicate detection
