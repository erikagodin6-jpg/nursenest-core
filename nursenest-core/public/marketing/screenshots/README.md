# Marketing screenshots (local capture output)

PNG outputs from `npm run capture:marketing-screenshots` (`scripts/capture-marketing-screenshots.mjs`) land here:

- `screenshot1.png` … `screenshot15.png` — maps to DigitalOcean Spaces root keys (see `docs/SCREENSHOT_CAPTURE_TO_CDN.md`).
- `capture-manifest.json` — run metadata, routes, themes, manual-review notes.
- `supplementary/` — optional non-slot captures.

Do **not** upload PNGs to Spaces until manually approved.
