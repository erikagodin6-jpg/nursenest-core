# Diagram exports

This audit used **Mermaid** sources in `IA_MERMAID.md` (no Figma MCP export in this run).

## Export to PNG/SVG

1. Use [Mermaid Live Editor](https://mermaid.live) — paste a fenced block from `IA_MERMAID.md`, export SVG/PNG.
2. Or VS Code “Markdown Preview Mermaid Support” extension → print to PDF.
3. **FigJam / Figma:** load `figma-generate-diagram` skill, then MCP `generate_diagram` with equivalent structure; save frames to `../figma-exports/` (create folder when using Figma).
