## Frame descriptions

**RN (Ocean canonical):** NCLEX-RN readiness hero; readiness score, projected CAT pass band, exam countdown; multi-hue system mastery bars (weak CV → strong endocrine/neuro); adaptive NGN case + continue lesson; quick-launch with **ECG workstation first**, then CAT, pharm, skills; integrated **labs / OSCE / transfusion** rail; premium analytics strip; study streak + recent activity; four NCLEX category cards.

**RPN:** Practical-nursing hero; metrics tuned to PN context; **skills lab, med admin, labs** dominate quick-launch with **ECG basics** last; OSCE + continue rail; skills-weighted analytics; PN-relevant categories.

**NP:** AGNP workstation hero; **four-up diagnostic density strip** (dx probability, rx flags, red labs, care gap); pharm-forward mastery row; quick-launch leads **pharm differential**, then advanced ECG, critical labs, OSCE; reasoning-weighted analytics; board blueprint categories.

**Themes:** Ocean = cool surfaces; Midnight = dark page + light type (same section order); Blossom = warm blush surfaces (same auto-layout).
