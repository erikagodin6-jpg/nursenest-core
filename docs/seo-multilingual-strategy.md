### A4: NOINDEX Rules by Language

| Rule | Threshold | When to Remove |
|---|---|---|
| Title must be translated | Required | When title_i18n[lang] exists |
| Body content must be translated | Required (>=80% of sections) | When 4 of 5 main sections translated |
| Meta title must be translated | Required | When meta_i18n.title[lang] exists |
| Meta description must be translated | Required | When meta_i18n.description[lang] exists |
| FAQ schema (if exists on EN) | Recommended | When faq_i18n[lang] has >=3 items |
| Overall completeness formula | title + body + meta_title + meta_desc = 4 fields. Need 4/4 = 100% for critical fields + 80% body sections | Automatically index when threshold met |
| Page types always noindex until complete | All lesson pages, hub pages, blog posts | Auto-index at threshold |
| Landing pages (flashcards, questions, pricing) | Can index with translated title/desc even if body is template | When title + desc translated |
