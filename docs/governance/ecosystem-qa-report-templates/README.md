# Ecosystem QA report templates

Copy a template to `reports/ecosystem-audit/` with a date prefix, for example:

`reports/ecosystem-audit/2026-05-09-ecosystem-qa-audit.md`

Fill **Failures**, **Warnings**, **Screenshots**, **Affected routes/pathways**, **Severity**, **Remediation**.

Guidance: `docs/governance/ecosystem-qa-master-program.md`
