# Lab Values — educational depth requirement

**Status:** Product + content directive for `/app/labs`  
**Cursor rule:** `.cursor/rules/lab-values-educational-depth.mdc`  
**Ecosystem:** `.cursor/rules/ecosystem-platform-guardrails.mdc`

## Purpose

Labs must feel like **adaptive clinical interpretation training**: reasoning, patterns, trends, prioritization, and bedside significance — **not** “memorize this normal range” or static reference tables.

## Instructional must-haves (per lesson where applicable)

- How labs **relate to each other** and what **combined** abnormalities suggest  
- Clinical context, nursing implications, prioritization  
- **Trend** significance and deterioration cues  
- Medication implications  
- ECG/telemetry correlations when clinically relevant  
- NGN-style interpretation prompts (prioritization, judgment) in micro-scenarios / questions  

## Tier scaling

- **RN:** Broad interpretation, prioritization, NCLEX readiness  
- **PN/RPN:** Bedside recognition, escalation, practical stability awareness  
- **NP:** Advanced diagnostic reasoning, deeper pathophysiology and trends  
- **Allied:** Profession-aware interpretation  
- **New Grad:** Specialty/unit context, ICU/ER/telemetry integration (via pathway/content, same routes)

## Content exemplars (integrated reasoning)

Hgb + Hct; viral vs bacterial patterns; neutrophilia; bandemia; left vs right shift; sepsis indicators; dilutional states; renal injury progression; electrolytes + ECG; acid-base; inflammatory markers; liver patterns; coagulation; cardiac marker progression.

## Integration

Wire into ECG, telemetry, medication safety, NGN, branching simulations, OSCE, adaptive remediation, and mastery/report surfaces using **existing** study loops and navigation — no isolated lab app.
