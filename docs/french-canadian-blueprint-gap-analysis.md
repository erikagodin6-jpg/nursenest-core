## Cross-Pathway Specialty Depth Requirements

Every French Canadian pathway must meet the specialty depth standard in `docs/blueprint-specialty-depth-requirements.md`, including:

- Adult medical-surgical nursing,
- pharmacology,
- pediatrics,
- maternal-newborn,
- mental health,
- critical care and emergency nursing,
- leadership and management,
- community and public health,
- professional practice and ethics,
- laboratory interpretation,
- ECG/cardiac monitoring where applicable,
- clinical skills,
- simulations.

## Blueprint Mapping Standard

Every French content item must map to:

- exam pathway,
- profession,
- body system,
- specialty,
- competency,
- question type where applicable,
- cognitive level,
- difficulty,
- clinical judgment phase,
- regulatory/scope note if relevant,
- adaptive remediation target.

## Highest-Risk Missing Domains

1. French Canadian regulatory/scope content.
2. French clinical judgment item types.
3. French rationales and distractor rationales.
4. French flashcards with clinical pearls and memory anchors.
5. French Canadian SEO content that matches actual learner search intent.
