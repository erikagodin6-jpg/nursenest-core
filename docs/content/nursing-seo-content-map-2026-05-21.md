# Nursing SEO Content Map - 2026-05-21

This map prevents duplicate or cannibalized publishing across NurseNest blog clusters.

## Governance Rules

- Publish one canonical page per primary search intent.
- If title, slug, primary keyword, and search intent overlap more than roughly 40-50%, update or expand the existing article instead of creating a new one.
- ECG, NCLEX/NGN, and NP articles must link vertically to their pillar and horizontally to narrow supporting articles.
- Images are optional for blog publishing; when used, prefer optimized CDN WebP/AVIF and stable dimensions.
