# French Canadian Flashcard Roadmap

Date: 2026-05-31

Status: `DRAFT - NOT READY FOR PUBLICATION`

## Flashcard Contract

Every French Canadian flashcard must include:

- prompt,
- answer,
- clinical pearl,
- memory anchor,
- exam relevance,
- common mistake,
- pathway,
- blueprint domain,
- body system,
- difficulty,
- review status,
- publication status.

One-word or term-only flashcards are draft assets and cannot count toward launch readiness.

## French Canadian Flashcard Example

```text
Prompt:
Chez une personne atteinte d'insuffisance cardiaque, quelle tendance doit etre signalee rapidement?

Answer:
Une prise de poids rapide, surtout si elle s'accompagne de dyspnee, d'orthopnee, d'oedeme ou de crepitants.

Clinical Pearl:
La prise de poids precede souvent l'aggravation visible de l'oedeme ou de la dyspnee severe.

Memory Anchor:
Poids qui monte = liquide qui s'accumule = poumons a surveiller.

Exam Relevance:
Les questions REx-PN et NCLEX testent la tendance, pas une seule mesure isolee.

Common Mistake:
Attendre que les chevilles soient tres enflees avant d'escalader.
```

## Review Gates

Flashcards cannot publish until:

- clinical reviewer approves the clinical claim,
- French Canadian language reviewer approves terminology,
- exam blueprint mapping is complete,
- card is linked to at least one lesson or question pool,
- duplicate detection passes,
- memory anchor is useful rather than cute or vague.
