# French Canadian Lesson Inventory Roadmap

Date: 2026-05-31

Status: `DRAFT - NOT READY FOR PUBLICATION`

## Lesson Quality Standard

French Canadian lessons must be written as original French Canadian nursing education, not literal translations. Each lesson must include:

- objectifs d'apprentissage,
- raisonnement clinique,
- evaluation infirmiere,
- interventions,
- alertes de securite,
- documentation,
- declencheurs d'escalade,
- perles cliniques,
- pieges d'examen,
- controles de connaissances,
- resume.

Target length:

| Pathway | Target length |
|---|---:|
| REx-PN | 1,500-2,500 words |
| Canadian NCLEX-RN | 1,500-3,000 words |
| CNPLE | 2,500-5,000 words |

## REx-PN Lesson Roadmap

Target: 500 French Canadian lessons.

Current source base: 427 English/source `ca-rpn-rex-pn` lessons.

Build sequence:

| Phase | Lesson group | Target count | Notes |
|---|---:|---:|---|
| 1 | Practical nursing safety, scope, delegation, documentation | 60 | Highest French Canadian differentiation; must reference Canadian scope without overclaiming province-specific law. |
| 2 | Adult med-surg core systems | 160 | Cardiovascular, respiratory, neuro, GI, endocrine, renal, infection, wounds, pain, mobility. |
| 3 | Pharmacology and medication safety | 90 | High-alert meds, insulin, anticoagulants, opioids, antimicrobials, psych meds, cardiac meds. |
| 4 | Maternal-newborn and pediatrics | 70 | RPN-relevant assessment, escalation, family teaching, medication safety. |
| 5 | Mental health, gerontology, community/LTC | 70 | High relevance for Canadian practical nursing work settings. |
| 6 | Exam strategy and clinical judgment | 50 | REx-PN item types, prioritization, stable vs unstable, cue recognition. |

Minimum first production batch: 50 lessons.

Batch 1 should include:

- Introduction au REx-PN pour les candidates et candidats francophones au Canada
- Priorisation clinique pour les infirmieres auxiliaires autorisees
- Patient stable ou instable: raisonnement pratique pour le REx-PN
- Delegation et collaboration avec les preposes aux beneficiaires
- Documentation infirmiere securitaire en contexte canadien
- Reconnaissance de la deterioration respiratoire
- Insuffisance cardiaque: poids quotidien, dyspnee et escalade
- MPOC: oxygenotherapie, dyspnee et securite
- Hypoglycemie et insulinotherapie
- Sepsis chez la personne agee en soins de longue duree

## French Canadian Terminology Guardrails

Use French Canadian clinical language where appropriate:

| Concept | Preferred French Canadian framing |
|---|---|
| Licensed practical/practical nurse | infirmiere auxiliaire autorisee / infirmier auxiliaire autorise where province/context supports it |
| RN | infirmiere autorisee / infirmier autorise |
| NP | infirmiere praticienne / infirmier praticien |
| Assessment | evaluation, evaluation infirmiere, collecte de donnees according to scope |
| Escalation | escalade, signalement rapide, avis a l'infirmiere responsable/prescripteur as context requires |
| Delegation | delegation, assignation, collaboration selon le champ d'exercice |
| Documentation | documentation, notes infirmieres, dossier clinique |
| Patient safety | securite du patient, prevention des dommages |

Avoid European-only terminology unless clinically standard in Canadian French.
