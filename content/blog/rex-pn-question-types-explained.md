# REx-PN Question Types Explained

The REx-PN uses multiple nursing question formats designed to evaluate safe practical nursing care.

Students commonly encounter:
- multiple-choice questions,
- SATA questions,
- prioritization,
- delegation,
- and clinical judgment scenarios.

Because the exam uses computerized adaptive testing (CAT), question difficulty changes throughout the exam.

Students preparing for the REx-PN should prioritize:
- adaptive practice questions,
- rationale review,
- and clinical reasoning development.
